# Politique de Sécurité

Nous prenons la sécurité de notre application très au sérieux. Si vous découvrez une faille de sécurité, merci de suivre la procédure ci-dessous pour nous la signaler de manière responsable.

## Versions Supportées

Seules les versions répertoriées ci-dessous reçoivent actuellement des correctifs de sécurité :

| Version | Supporté |
| ------- | --------- |
| main    | ✅ Oui    |
| < v1.0  | ❌ Non    |

## Signalement d'une vulnérabilité

**Merci de ne pas ouvrir de ticket public (GitHub Issue) pour signaler une faille de sécurité.**

Pour signaler une vulnérabilité :
1. Envoyez un e-mail détaillé à [contact@owlinstack.com](mailto:contact@owlinstack.com).
2. Ou utilisez l'outil de **Signalement privé de vulnérabilité** (*Private Vulnerability Reporting*) directement dans l'onglet **Security** du dépôt GitHub.

Nous nous efforçons de répondre sous 48 heures et de proposer un correctif dans les plus brefs délais. Merci de nous accorder le temps nécessaire pour corriger la faille avant toute divulgation publique.
